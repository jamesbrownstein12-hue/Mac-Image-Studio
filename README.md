Mac Image Studio  

This app was designed with Codex and contains so far, 12 different workflows with diverse file conversion needs.  

Workflows so far:  
1: MacOS Installer Apps to DMG, ISO, CDR, and IMG  

2: MacOS InstallAssistant.pkg's to the standard APP, or convert to DMG, ISO, CDR, and IMG  

3: PKG Extracter, will extract contents of a pkg and puts the application from the payload into whatever folder you choose.   

4: USB Flasher, flashes MacOS InstallAssistant.pkg's after extracting, MacOS Install Apps, DMG, ISO, CDR, IMG, QCOW2, VDI, VMDK, VHD, and VDHX    

5: Audio Converter: Can convert MP3, AAC, M4A, WAV, FLAC, AIFF, OGG, and OPUS. Requires ffmpeg tool from Homebrew but the app should give you an option to download it if you don't have it yet.  

6: Video Converter: Can convert MP4, MOV, MKV, AVI, WEBM, M4V, MPG, and TS. Requires ffmpeg tool from Homebrew but the app should give you an option to download it if you don't have it yet.    

7: Image Converter: Can convert PNG, JPG, TIFF, HEIC, BMP, GIF, and ICNS. Doesn't require any external tools and uses built in MacOS Stuff.  

8: Archive Converter: Can Convert ZIP, 7Z, TAR, TAR.GZ, and TAR.XZ. Requires 7z tool from Homebrew but the app should give you an option to download it if you don't have it yet.  

9: Unarchiver: Can choose any archive format and will extract into a folder. Requires 7z tool from Homebrew but the app should give you an option to download it if you don't have it yet.  

10: Virtual Disk Converter: Can convert QCOW2, VDI, VMDK, VHD, and VDHX to DMG, ISO, CDR, and IMG. Requires the QEMU-IMG tool from Homebrew but the app should give you an option to download it if you don't have it yet.  

11: Youtube Downloader: Paste in a youtube link, and you will be able to download a diverse number of video file options and resolutions. You will need the yt-dlp excecutable tool, but the app should give you an option to download it if you don't have it yet.  

12: App Signer: You can choose an .app file, and you can either sign the app with the xattr quarantine, or codesign the app with codesign.  

Hope you enjoy and more workflows coming in the future.
